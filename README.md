# ASU Report Converter

Конвертер отчётов из CSV в Excel с графическим интерфейсом (PySide6).

## Возможности

- Выбор CSV-файла через диалог
- Чтение CSV: разделитель `;`, кодировка UTF-8
- Удаление полностью пустых столбцов
- Автоматическое приведение столбцов к числовому типу (где возможно)
- Сохранение результата в Excel (`.xlsx`) в ту же папку, что и исходный файл

## Требования

- Python 3.x
- PySide6
- pandas
- openpyxl

## Установка

```bash
pip install PySide6 pandas openpyxl
```

## Запуск

```bash
python main.py
```

## Использование

1. Нажмите **«Выбрать отчет для конвертации»** и укажите CSV-файл.
2. Нажмите **«Пуск»**.
3. Файл Excel будет создан рядом с выбранным CSV с тем же именем и расширением `.xlsx`.

## Структура проекта

- `main.py` — приложение с GUI и логикой конвертации
- `converter.py` — отдельный скрипт для просмотра RTSP-потока (не связан с конвертацией отчётов)
